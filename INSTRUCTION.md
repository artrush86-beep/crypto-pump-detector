# 🤖 Crypto Pump Detector - Инструкция

## Команды бота

### Основные команды:
- `/start` — Запуск бота, показывает приветствие и доступные команды
- `/help` — Помощь: объяснение сигналов и метрик
- `/status` — Статус бота: пороги, интервал сканирования
- `/settings` — Настройки (кнопки меню)
- `/alert SYMBOL %` — Создать ценовой алерт
- `/ignore SYMBOL` — скрыть пару из новых уведомлений
- `/unignore SYMBOL` — вернуть пару в уведомления
- `/ignored` — показать скрытые пары
- `/stop` — поставить сканирование на паузу
- `/resume` — продолжить сканирование
  - Пример: `/alert BTCUSDT 1` — уведомит при изменении цены на 1%
  - Пример: `/alert ETHUSDT 2.5` — уведомит при изменении на 2.5%

## 📊 Что означают сигналы?

### Типы сигналов:
- 🛰️ **EARLY LONG / SHORT WATCH** — раннее предупреждение до расширения цены
- 🚀 **LONG SIGNAL** — подтверждённый импульс вверх
- 🔻 **SHORT SIGNAL** — подтверждённый импульс вниз

### Таймфреймы:
- **5м** — быстрые сигналы по реальному окну 5 минут
- **15м** — стандартные сигналы по окну 15 минут
- **1ч** — сильные сигналы по окну 60 минут

### Скор (Score):
- **5/5** 🔴 EXTREME — очень сильный сигнал
- **4/5** 🟠 HIGH — сильный сигнал
- **3/5** 🟡 MEDIUM — умеренный сигнал

---

## 📈 Метрики в сигнале

| Метрика | Описание |
|---------|----------|
| **OI** | Open Interest — изменение открытого интереса (сколько денег в позициях) |
| **Цена** | Изменение цены за выбранный таймфрейм |
| **Объём** | Рост объёма торгов против обычного потока в этом окне |
| **Фандинг** | Ставка фандинга: отрицательная = шорты платят, положительная = лонги платят |
| **L/S** | Long/Short Ratio — соотношение лонгов к шортам |

## ⚙️ Настройка порогов

Пороги меняются через переменные окружения в Railway:

| Переменная | Описание | Значение по умолчанию |
|------------|----------|----------------------|
| `OI_CHANGE_THRESHOLD` | Порог изменения OI | 5.0% |
| `PRICE_CHANGE_THRESHOLD` | Порог изменения цены | 1.0% |
| `VOLUME_CHANGE_THRESHOLD` | Порог роста объёма | 50.0% |
| `MIN_SIGNAL_SCORE` | Минимальный скор подтверждённого сигнала | 3 |
| `EARLY_SIGNAL_MIN_SCORE` | Минимальный скор раннего сигнала | 2.5 |
| `SCAN_INTERVAL` | Интервал сканирования | 300 сек |

## 💡 Рекомендации по использованию

### Для трейдинга:
1. **EARLY** — использовать как watchlist и подготовку к сделке
2. **CONFIRMED** — использовать как подтверждение направления
3. **Таймфрейм 5м** — для быстрых входов и ускорения рынка
4. **Таймфрейм 15м** — для внутридневной торговли
5. **Таймфрейм 1ч** — для более сильных и редких движений

### Алерты:
- Ставьте алерты на монеты из сигналов
- 1% для волатильных монет
- 2-3% для стабильных монет

### Риск-менеджмент:
- Сигналы — это индикаторы, не гарантия прибыли
- Всегда используйте стоп-лоссы
- Не входите на всё доступное плечо
- Диверсифицируйте позиции
